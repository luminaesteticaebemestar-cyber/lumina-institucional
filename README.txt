Lumina — HTML Pro Pack

Conteúdo:
- index.html (otimizado, com OG/Twitter, JSON-LD, lazy-loading, CTA fixo mobile)
- /images (renders otimizados + favicon.png)
- robots.txt
- sitemap.xml

Publicação rápida:
1) GitHub Pages: suba tudo numa branch e ative Pages. URL: https://<usuario>.github.io/<repo>/
2) Netlify: arraste a pasta para o painel. Pronto.
3) Vercel: importe do GitHub em 1 clique ou arraste a pasta.

Ajustes:
- Troque o domínio em robots.txt e sitemap.xml por seu domínio real.
- Edite os CTAs (WhatsApp/e-mail) em index.html se quiser alterar o texto.
